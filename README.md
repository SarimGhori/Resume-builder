# Resume Builder

A modern Resume Builder web application developed using Laravel. Users can create, manage, and download professional resumes easily.

## Features

- User Registration & Login
- Profile Management
- Education Section
- Experience Section
- Skills Section
- Resume Preview
- Responsive Design

## Technologies Used

- Laravel
- PHP
- MySQL
- Blade Templates
- Bootstrap
- HTML
- CSS
- JavaScript
